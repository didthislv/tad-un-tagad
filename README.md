# tad-un-tagad
“Tad un Tagad” ir intuitīva aplikācija, kas ļauj salīdzināt vēsturiskus un mūsdienu attēlus vienuviet. Lietotāji var redzēt, kā laika gaitā mainījušās pilsētas, ēkas un ainavas, pārvelkot slīdni starp “toreiz” un “tagad”. Aplikācija veicina interesi par vēsturi un palīdz labāk izprast apkārtējās vides attīstību.
